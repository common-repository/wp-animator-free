== Description ==

WordPress animator free :  This is a great tool for the realization of your creative ideas. With its help you can attract customers and make your original site.  You can easily positiont the elements on any part of the screen, for example - links, images, banners, switches, running lines, etc.  and  to apply animation and  tooltip to it. ### The Plugin Features:

* More than forty-five CSS animations
* Eleven tooltips styles
* Display conditions on different pages
* Hover effect to start and stop animation
* Play the second animation.
* The ability to add animation to any element of the page.
* The ability to hide elements outside the container.
* Start animation when the element on the screen
* Generating CSS styles ( 1-file ) to save your server resources
